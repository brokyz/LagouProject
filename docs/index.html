    关于项目

    本项目集合了爬虫, 数据分析, 数据可视化, 静态网页多种实现手段

    爬虫部分, 本项目的数据均爬自Boss直聘网站

    数据分析部分, 本项目将爬取到的数据做简单的数据分析

    数据可视化部分, 本项目将分析得到的数据, 利用开源项目pyecharts, 进行数据可视化

    静态网页部分, 本项目将部分数据导入SQLiet以供静态网站读取, 并将数据可视化部分得到的图表在静态网页进行展示, 最后利用github page项目将静态网页映射到公网

    -

    本项目所有源码均已在 Github 进行开源
